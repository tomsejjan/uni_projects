Detailed Description

Behavioral architecture of the generic entity synchronous binary counter with output q[BITS-1:0] with limitation of counting to MAX.

cnt_proc ( clk  )

Process handling the counter increment logic.

When ce = 0, the current value of cnt_i is maintained. When ce = 1 and the current state is less than MAX, the counter increments. If the maximum value MAX is reached, the counter resets to 0.
